# commandBased2023

commandBased2023 is the base code for Team 5442's 2023 robot. It has been proven and tested and will work. 
 
**Current Features:**
 
>* Feild-Oriented Swerve
>* Optimized Inverse Kinematics 
>* Tunable Drivetrain
>* Angular Positioning 
>* Basic Vision
>* Odometry
>* Configurable Autonomous Code
